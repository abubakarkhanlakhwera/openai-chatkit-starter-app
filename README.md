# OpenAI ChatKit Starter App

A Next.js starter app for experimenting with OpenAI ChatKit and agent-powered chat UI.

## Purpose

This repository is for quickly exploring a hosted ChatKit-style chat interface. It helps test the shape of an AI chat product before building a larger application around it.

## What This Shows

- Next.js chat UI structure
- OpenAI ChatKit integration
- Session endpoint and frontend wiring

## Stack

- Next.js
- TypeScript
- OpenAI ChatKit
